# zhihu
高仿知乎客户端

####第一次比较完整的App，还处于学习中。。。

### 已经完成的功能
 
 + 新闻页面的数据展示
 + 新闻的详细页面展示
 + 可以上下滑动换页
 + 点击链接可以打开新的页面
 + 上拉跟下拉刷新数据
 + 抽屉导航栏
 + 启动动画的展示
